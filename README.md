# borck.education

AI-powered tools for educators — privacy-first, locally-run, and built for the classroom.

## Featured Tools

- **curriculum-curator** — AI-powered course content creation
- **talk-buddy** — AI conversation practice with speech
- **study-buddy** — Desktop AI tutoring with local inference
- **feed-forward** — AI formative feedback on assignments
- **hands-on-ai** — Educational AI toolkit with chatbot, RAG, and agent modules
